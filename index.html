Projects

Virtual Painting System using Finger Gesture Detection and Recognition (Link to Project)

Relevant Skills: Python, OpenCV, MediaPipe, NumPy, Computer Vision, Gesture Recognition, Real-time Processing

Developed a contactless virtual painting system that translates finger gestures into real-time brush strokes. The system performs hand landmark detection, fingertip tracking and gesture-to-command mapping to support drawing, erasing certain parts on screen and clearing canvases. Implemented smoothing and low-latency optimizations for a responsive experience. The project demonstrates practical applications of computer-vision-based interaction for creative tools and accessibility.

CAPTIONARY-Generating Image Descriptions With RESNET AND LSTM (Link to Project)

Relevant Skills: Python, Tensorflow, ResNet, LSTM, Image Captioning, Flickr8k Dataset, Feature Extraction, Model Training

Designing and implementing an image captioning model combining ResNet for deep visual feature extraction and an LSTM decoder for natural language generation. Processed and cleaned the Flickr8k dataset, and embedded captions using tokenization and sequence padding. Trained the CNN-LSTM architecture to generate meaningful textual descriptions of images. The project demonstrates the integration of computer vision and natural language processing for multimodal learning.
